# The Baby Mod 🍼👶

A fun and hilarious Minecraft Fabric mod for version 1.21.1 that adds a **Baby Rattle item** with amazing transformation mechanics and protective powers!

## Overview

Turn yourself and mobs into adorable babies with special powers! Get protection, unlock secret abilities, and dominate the Minecraft world in baby form. This mod adds a completely new gameplay experience centered around the power of being a baby!

---

## 🎮 Features

### 🍼 **Baby Rattle Item**
- A gun-like item that shoots transformation beams
- Craft it or find it in creative mode
- Right-click to activate transformation beam
- Infinite uses - never breaks!

### 👶 **Transformation Mechanic**
Transform yourself and mobs into cute babies:
- **Custom Baby Skin** - Adorable baby player skin with diaper overlay
- **Proportional Head** - Head is 2 pixels bigger than body (baby-like appearance)
- **Custom Baby Rendering** - Proper baby model with animations
- **Smooth Transformation** - Animated transformation effect
- **Revert Anytime** - Transform back to normal form

### 🛡️ **Baby Protection System**
When you're a baby, you gain incredible defensive powers:
- **Invincible** - Take zero damage from any source
- **Attacker's Fate** - Any mob or player that attacks you instantly dies
- **Mob Immunity** - No mobs will attack you while in baby form
- **Crying Animation** - Baby cries for protection (visual effect)
- **Ultimate Safety** - Complete immunity to all harm!

### 🍶 **Baby Bottle Projectile**
- Summon a magical baby bottle from the sky
- The bottle crashes down and kills everything in its path
- Useful for mob farms and defense
- Spectacular visual effect

### 😭 **Burping Mechanic**
- Babies burp periodically while transformed
- Authentic baby sounds included
- Adds immersion to the baby experience
- Can be toggled on/off

### 🍽️ **Feeding System**
- Feed the baby various foods to restore health
- Gain temporary buffs from feeding:
  - Strength boost
  - Speed increase
  - Regeneration effect
- Keep your baby happy and healthy!

### 🏠 **Home Teleport Trap**
- Bring attackers to your home dimension
- They'll be transported to a special trap area
- Ultimate defense mechanism
- Escape dangerous situations instantly

## 📋 Installation Guide

### Prerequisites
- Minecraft Java Edition (1.21.1)
- Fabric Loader 1.21.1
- Java 21+ installed

### Step 1: Install Fabric Loader
1. Download Fabric Installer from [fabricmc.net](https://fabricmc.net)
2. Run the installer
3. Select Minecraft version **1.21.1**
4. Click **"Install"**

### Step 2: Install Fabric API
1. Download Fabric API 1.21.1 from [modrinth.com/mod/fabric-api](https://modrinth.com/mod/fabric-api)
2. Place the JAR in your `mods` folder:
   - Windows: `%APPDATA%\.minecraft\mods`
   - Mac: `~/Library/Application Support/minecraft/mods`
   - Linux: `~/.minecraft/mods`

### Step 3: Install Baby Mod
1. Download `baby-mod.jar` from the releases folder
2. Place it in your `mods` folder
3. Launch Minecraft with the **Fabric 1.21.1** profile
4. Enjoy! 🎉

## 🎮 How to Play

### Getting Started
1. Launch Minecraft with Fabric loaded
2. Create a new world or join an existing one
3. Open creative mode or use `/give @s baby_rattle` to get the item

### Controls & Actions
- **Right Click** - Shoot transformation beam at yourself or mobs
- **R Key** - Toggle baby form (if in range)
- **Hold & Aim** - Aim the rattle at targets
- **Shift + Right Click** - Special attack (baby bottle projectile)
- **Sneak + Interact** - Feed the baby (when near food)

### Gameplay Tips
- 🎯 Use the transformation beam to turn enemies into harmless babies
- 🛡️ Transform yourself for ultimate protection in tough situations
- 🍶 Summon baby bottles for crowd control
- 🏠 Use home teleport as a last resort escape mechanism
- 😭 Listen for burps - they indicate your baby status is active

## 🔧 Building from Source

### Requirements
- Git
- Java Development Kit (JDK) 21+
- Gradle (included via wrapper)

### Build Steps

```bash
# Clone the repository
git clone https://github.com/koolkainsley-ops/The-baby-mod.git
cd The-baby-mod

# Build the mod
./gradlew build

# The JAR will be in: build/libs/baby-mod-1.0.0.jar
```

### File Structure
```
The-baby-mod/
├── src/
│   └── main/
│       ├── java/          # Java source code
│       └── resources/      # Textures, sounds, configs
├── build.gradle           # Build configuration
├── gradle/                # Gradle wrapper
└── README.md              # This file
```
