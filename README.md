# The Baby Mod

A fun Minecraft Fabric mod for version 1.21.1 that adds a Baby Rattle item with transformation mechanics!

## Features

🍼 **Baby Rattle Item** - A gun-like item that shoots transformation beams

👶 **Transformation Mechanic** - Turn yourself and mobs into cute babies:
- Baby player skin with diaper overlay
- Head is 2 pixels bigger than body
- Custom baby rendering

🛡️ **Baby Protection** - When you're a baby:
- Take no damage - attackers instantly die instead
- Mobs won't attack you
- Crying animation for protection
- No one can hurt you!

🍶 **Baby Bottle Projectile** - Summon a baby bottle from the sky that kills everything in its path

😭 **Burping Mechanic** - Babies burp periodically (with sound!)

🍽️ **Feeding System** - Feed the baby to restore health and gain buffs

🏠 **Home Teleport Trap** - Bring attackers to your home dimension where they meet their doom

## Installation

1. Download Fabric Loader 1.21.1 from [fabricmc.net](https://fabricmc.net)
2. Install Fabric API 1.21.1
3. Place this mod JAR in your `mods` folder
4. Launch with Fabric profile

## Controls

- **R Key** - Transform with Baby Rattle
- **Right Click with Baby Rattle** - Shoot transformation beam

## Building from Source

```bash
git clone https://github.com/koolkainsley-ops/The-baby-mod.git
cd The-baby-mod
./gradlew build
```

The compiled JAR will be in `build/libs/`
